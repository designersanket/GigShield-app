🛡️ GigShield

AI-Powered Freelance Protection System

🚀 Inspiration

Freelancing is growing rapidly, but so are scams — fake clients, unpaid work, and misleading job descriptions.

As students and developers, we realized there is no intelligent system that actively protects freelancers before they accept a gig.

That’s where GigShield comes in — an AI layer of trust designed to analyze, detect, and prevent risky opportunities.

💡 What it does

GigShield acts as a smart safety assistant for freelancers:

🔍 Detects scam patterns in job descriptions

🤖 Uses AI (LLM) to analyze gig content

📊 Generates a risk score (Low / Medium / High)

⚠️ Highlights suspicious phrases and red flags

💬 Provides AI-powered suggestions

🧠 Helps users make safe, informed decisions

🛠️ How we built it
⚙️ Tech Stack

Frontend: React (Vercel)

Backend: Node.js + Express (Render)

Database: MongoDB Atlas

AI Engine: Google Gemini API

🧩 Workflow

User inputs gig/job description

Backend sends structured prompt to Gemini API

AI analyzes risk factors

Returns:

Risk Score

Explanation

Warning signals

Frontend displays insights in a clean dashboard
